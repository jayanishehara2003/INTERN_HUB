# 🚀INTERN_HUB

InternHub is a MERN stack web application that helps students learn, practice, and find internship opportunities—all in one place.

## ✨ Features

🔐 Secure Authentication

* User Registration & Login
* JWT-based Authentication

📚 Study Materials

* Access learning resources and guides
* Improve technical and professional skills

📝 Interactive Quizzes

* Test your knowledge
* Track your learning progress

💼 Internship Vacancies

* Browse available internship opportunities
* View vacancy details and requirements

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* Authentication: JWT

## 🎯 Purpose

InternHub bridges the gap between learning and career opportunities by providing students with the resources, assessments, and internship listings they need to grow professionally.

## 🌟 Built With MERN

MongoDB • Express.js • React.js • Node.js
